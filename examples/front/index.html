js 定时器实现动画效果

利用定时器每隔16ms刷新一次网页内容和样式。可以实现较复杂的效果，但性能消耗较大。

关键点：setInterval,requestAnimationFrame函数

css transition (过渡)

使元素从一个状态到另一个状态的过程变得平缓，一般用于实现简单动画效果。
